Convert PHP array to human friendly format

CONVERT SHOW TEXTAREA 
	请在自己编写的程序里，把var_export($arr)输出的结果粘贴至文本域中，形如：
	var_export(array ('a' => 'apple', 'b' => 'banana', 'c' => array ('x','y','z')));

	会输出这样的结果，将以下内容粘贴至文本域中：
	array ( 'a' => 'apple', 'b' => 'banana', 'c' => array ( 0 => 'x', 1 => 'y', 2 => 'z', ), )
