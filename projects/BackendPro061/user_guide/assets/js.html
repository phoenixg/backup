Javascript

BackendPro 通篇都是精简的脚本，提供给用户自己完善功能.

Note:  本页讨论的所有脚本都在BackendPro的 Page类里自动加载.

全选复选框 Multiple Check Checkboxes

BackendPro 通篇的复选框，能选择并执行任务. 为了让用户更方便地选择所有的项目，可以创建一个复选框，选择它时所有复选框都被选中. 要使用这个功能，无需用到任何的avascript脚本,你只需用以下方式指定复选框:

<form>
    <input type="checkbox" name="all" value="myCheckbox">
    <input type="checkbox" name="myCheckbox[]" value="1"> 
    <input type="checkbox" name="myCheckbox[]" value="3"> 
    <input type="checkbox" name="myCheckbox[]" value="2"> 
</form>

如你所见，有一个叫all的复选框。它的值就等于它要控制的那些复选框，就这么简单.
