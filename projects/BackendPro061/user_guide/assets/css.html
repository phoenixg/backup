CSS 样式

表单、表格、链接等的样式模板.

Note:  本页讨论的所有样式默认由BackendPro的 Page 类自动加载.

表单

BackendPro的全部表单样式几乎都是一致的. 我提供了一致的html代码，无论是横向还是纵向的表单。 基本的表单样式如下:

<form class="form_type">
    <fieldset>
        <ol>
            <li>
                <label for="inpText">Label Text:</label>
                <input type="text" name="inpText" class="text">
            </li>
            .
            .
            .
            <li class="submit">
                <input type="submit" name="submit" value="Save">
            </li>
        </ol>
    </fieldset>
</form>

使用 ol 元素创建表单. 每行一个新 li. form_type 可以是 horizontal 或 vertical 两个值的任意一个。用于指定表单的定向.

再下面是一个 class 应用给input元素, 用于将text input的大小和select以及text area的大小一致,并非必须要使用它.

最后的li有一个class值是 submit, 告诉表单当在 horizontal 模式下就 indent提交按钮。要创建可爱的按钮，请见 Icon Buttons.

Note: 更多关于表单样式的信息见 assets/shared/css/forms.css.

Icon Links 图标链接

BackendPro 通篇你都会看见链接旁边有图标. 这是由PHP生成的样式表控制的，位于 assets/shared/css/icons.php.

每个图标都存储于 assets/shared/icons 下面，CSS代码会自动创建，所以可以被作为链接图标使用, 你只要使用以下格式即可:

<a href="location" class="icon_name">Some Link</a>

name 是你想要显示在链接旁边的icon文件的名称,不包含扩展名.

Icon Buttons 图标按钮

BackendPro 提供了简单单样式丰富的方法来创建HTML的 <button> 标签. 你再也不用创建难看的按钮，你可以添加有意思的图标愉悦你的用户.

Button示例代码:

<div class="buttons">
    <button type="submit" class="positive">
         <?= $this->page->icon('disk');?>
         Submit
    </button>
    
    <a href="#" class="negative">
         <?= $this->page->icon('cross');?>
         Cancel
    </a>
    
    <a href="#">
         <?= $this->page->icon('key');?>
         Change Password
    </a>
</div>

正如你在上面所见, 3个按钮就这么创建了. 你要么可以使用 <button> 标签来创建，要么可以使用 <a> 标签来创建. 通过应用 positive/negative 会相应调整按钮颜色为 green/red . 如果不指定一个 class ,就会显示一个中立按钮.

正如你在示例中所见, $this->page->icon('cross'); 被用来包含一个图标图片. 该函数的更多信息见 Page类库.
