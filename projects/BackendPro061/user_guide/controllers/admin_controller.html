Admin Controller Class 管理员控制器类

所有的admin控制器都应该继承这个控制器类, I.E 任何属于管理员区域的控制器. 该类可以在这里被找到 system/application/libraries/MY_Controller.php.

如何使用 Admin Controller Class

当创建你自己的管理员控制器时，像下面这样继承这个类:

class Control_panel extends Admin_Controller
{
         function Control_panel()
         {
             // Call parent constructor
             parent::Admin_Controller();
         }
}
