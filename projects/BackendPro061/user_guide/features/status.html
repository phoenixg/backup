Status Class

Status 类落实了一个简单干净的方法，用于在页面加载完毕后显示消息给用户.

Note:  该类由系统自动初始化，所以无需手动这么做.

Status Class 函数参考

flashMsg()

创建新的消息:

flashMsg('type','message')

第一个参数 type 指定了消息的类型. 可以是如下的任意一项, info, success, warning & error.

第二个参数 message 是你想要显示给用户的消息.

displayStatus()

显示存储的消息:

displayStatus('type')

第一个可选 参数指定了你想要输出的错误消息类型. 可以是这几个值 info, success, warning & error. 正如所说的,它是可选的,如果没有类型指定，那么全部错误消息类型会被输出.
