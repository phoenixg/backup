Install Process安装步骤

BackendPro 自带一个安装过程来安装系统.

Important: 安装之前请确保你的 CodeIgniter版本是最新的. BackendPro 只有在CodeIgniter已安装的情况下才能运行

开始安装

要开始安装，直接访问 /install 目录 ,你会看见表单和其他指示.

安装步骤

 

  • CodeIgniter 文件系统
    首先你被问及CI的文件系统的结构.如果你改变了application文件夹的路径，或是其他主要的system文件夹的路径，那么这个项目就很有用. 请注意这些文件夹不会在CI文件里被改变，你只能手动去改，这只是为了安装过程.
  • Database 数据库
    你被问及应用程序设置的数据库详情. 用户指定的必须把被创建为 InnoDB 类型并且指定的数据库必须已经被创建.
  • User Account 用户账号
    这是系统的第一个用户账号. 它被初始授予管理员权限，但你可以在接下来的安装过程中改变这个权限.
  • ReCAPTCHA
    请注意这个部分并非必填项目，除非你想在注册和登录页面使用captcha检查. 如果你决定不在这个阶段提供 public/private 钥你也可以手动在系统的文件里设置modules/recaptcha/config/recaptcha.php.
  • Encryption Key加密钥
    在安装期间会生成一个加密钥在你的system/application/config/config.php文件中. 不要在安装完毕后改变这个KEY. 这么做会毁掉当前系统所有用户的密码.

Important: 安装后请移除 /install 目录，以免安全隐患
